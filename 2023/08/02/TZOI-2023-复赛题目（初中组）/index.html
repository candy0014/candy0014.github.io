TZOI 2023 游记（初中组）

T1

平面上给定 nnn 个点 (xi,yi)(x_i,y_i)(xi​,yi​)。依次加入这些点，问：当加到第几个点时，存在 kkk 个点排成了一条直线，横着，竖着，斜着都可以。若最后都没有，输出 -1。

1≤k≤n≤1061 \le k \le n \le 10^61≤k≤n≤106，1≤xi,yi≤1051 \le x_i,y_i \le 10^51≤xi​,yi​≤105

T2

给定一个长度为 nnn 的序列 {ai}\{a_i\}{ai​}，问有多少对 (i,j)(i,j)(i,j) 满足 ai2≤aj≤ai\dfrac{a_i}{2} \le a_j \le a_i2ai​​≤aj​≤ai​，注意，iii 不一定小于 jjj。

1≤n≤1051 \le n \le 10^51≤n≤105，1≤ai≤1091 \le a_i \le 10^91≤ai​≤109

T3

给定一个 n×mn \times mn×m 的矩阵，问有多少个子矩阵满足子矩阵内所有元素相同。

1≤n,m≤1031 \le n,m \le 10^31≤n,m≤103

T4

给定一个长度为 nnn 的互不相同的序列 {ai}\{a_i\}{ai​} 以及一个整数 k∣nk\mid nk∣n，定义一个好序列为：将这 nnn 个数按每 kkk 个一段分组，每一组的和单调递增。

你现在可以做操作，一次操作可以把某个元素 aia_iai​ 取出，插入到序列的任意位置。问最少几次操作可以使得 a{a}a 数组成为一个好序列。

1≤n≤5×1031 \le n \le 5 \times 10^31≤n≤5×103
